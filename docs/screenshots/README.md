## How to take screenshots

1. Deploy the project following DEPLOYMENT.md
2. Open the live URL in a browser
3. For the 3D viewer screenshot: search TP53 R175H, wait for structure to load, zoom to a good angle
4. Use full-page screenshots at 1440px width for best quality
